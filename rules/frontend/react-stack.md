# React Stack

## Core Setup
- React with TypeScript
- Vite as build tool
- Tailwind CSS for styling
- Use `tsconfig.json` with strict mode

## File Structure
- Components in PascalCase: `UserProfile.tsx`
- Hooks in camelCase with `use` prefix: `useAuth.ts`
